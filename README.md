# Instagram Image Analyzer

Download Instagram images and prepare them for AI analysis.

## Features

- Extract URLs from Excel files
- Download images from Instagram
- Generate Gemini AI prompts for image analysis
- Merge analysis results with Excel data

## Setup

```bash
pip install -r requirements.txt
```

## Run locally

```bash
streamlit run app.py
```

## Deploy to Streamlit Cloud

1. Push to GitHub
2. Go to [streamlit.io/cloud](https://streamlit.io/cloud)
3. Connect repo and deploy

## Requirements

- Python 3.8+
- See requirements.txt for packages
